Get started

    API Endpoint

        http://127.0.0.1:27800/

AnalogAPI is a Fabric mod for Minecraft that exposes the player's movement state — and lets external programs drive the player — through a small HTTP API running on localhost.

The server binds to 127.0.0.1:27800 by default. You can override the host and port with the system properties analogapi.host / analogapi.port or with the environment variables ANALOGAPI_HOST / ANALOGAPI_PORT.

All endpoints accept and return application/json. CORS is fully open (Access-Control-Allow-Origin: *) so browser-based dashboards can call the API directly.

Health check


# Liveness probe
curl http://127.0.0.1:27800/api/health

A simple liveness probe that always returns 200 OK while the mod is loaded.
GET http://127.0.0.1:27800/api/health 


Result example :

{
  "ok": true,
  "name": "AnalogAPI"
}

Get movement state


# Read the latest computed movement
curl http://127.0.0.1:27800/api/state

Returns the latest movement state computed by the mod for the current client tick.
GET http://127.0.0.1:27800/api/state 


Result example :

{
  "forward":  0.85,
  "sideways": -0.32,
  "jump":     false,
  "sneak":    false,
  "sprint":   true
}

RESPONSE FIELDS

Field Type Description
forward Float Forward (+) / backward (−) impulse, in [-1.0, 1.0].
sideways Float Right (+) / left (−) impulse, in [-1.0, 1.0].
jump Boolean Whether the jump key is pressed this tick.
sneak Boolean Whether the sneak / shift key is pressed this tick.
sprint Boolean Whether the sprint key is pressed this tick.

Get virtual input


# Check whether a virtual input is currently driving the player
curl http://127.0.0.1:27800/api/virtual-input

Reports whether a virtual input is currently overriding the keyboard, and if so what its values are.
GET http://127.0.0.1:27800/api/virtual-input 


Inactive :

{ "active": false }

Active :

{
  "active":   true,
  "forward":  1.0,
  "sideways": 0.0,
  "jump":     false,
  "sneak":    false,
  "sprint":   true
}

Set virtual input


# Walk forward at full speed while sprinting
curl -X POST http://127.0.0.1:27800/api/virtual-input \
     -H "Content-Type: application/json" \
     -d '{"forward":1.0,"sprint":true}'

Set or replace the virtual input. While set, the mixin uses these values instead of the real keyboard / analog input.
POST http://127.0.0.1:27800/api/virtual-input (PUT is also accepted)

BODY PARAMETERS

Field Type Description
forward Float (optional — default 0) Forward / backward impulse, clamped to [-1.0, 1.0].
sideways Float (optional — default 0) Right / left impulse, clamped to [-1.0, 1.0].
jump Boolean (optional — default false) Whether to hold the jump key.
sneak Boolean (optional — default false) Whether to hold sneak.
sprint Boolean (optional — default false) Whether to hold sprint. 

Result example :

{ "active": true }

Clear virtual input


# Hand control back to the keyboard
curl -X DELETE http://127.0.0.1:27800/api/virtual-input

Removes any active virtual input and restores normal keyboard / libanalog behavior.
DELETE http://127.0.0.1:27800/api/virtual-input 


Result example :

{ "active": false }

Errors

AnalogAPI returns standard HTTP status codes. The body is always JSON.

Status Meaning
200 Request handled successfully.
400 Invalid JSON body on POST /api/virtual-input. The response includes {"error":"invalid_json","message":"…"}.
404 Unknown path. The response is {"error":"not_found"}.
405 HTTP method not supported on this endpoint. The response is {"error":"method_not_allowed"}.
